Address Book API

Welcome to the Jakarta EE Address Book API powered by OpenLiberty and Apache Derby.

API Endpoints

All endpoints are available under /api/addressbook

GET /api/addressbook
Get all contacts
GET /api/addressbook/{id}
Get contact by ID
POST /api/addressbook
Create new contact
PUT /api/addressbook/{id}
Update existing contact
DELETE /api/addressbook/{id}
Delete contact
GET /api/addressbook/search?name={name}
Search contacts by name

Contact Data Structure

{
  "id": 1,
  "name": "John Doe",
  "phoneNumber": "+1-555-0123",
  "address": "123 Main St, Anytown, USA"
}

Technologies Used

  • Jakarta EE 10
  • OpenLiberty
  • Apache Derby Database
  • Java 17
